# Corcho

Una aplicación de lista de tareas minimalista y eficiente.

## Descripción

El objetivo de este proyecto era crear una herramienta simple para la gestión diaria de tareas, enfocada en la rapidez y la usabilidad.

### Funcionalidades

- Crear tareas con prioridad.
- Funcionalidad Drag & Drop.
- Persistencia de datos en LocalStorage.
