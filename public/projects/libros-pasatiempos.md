# Libros Pasatiempos

Una colección de libros interactivos y digitales diseñados para el entretenimiento y el desafío mental.

## Sobre el Proyecto

Este proyecto nació de la idea de transformar los libros de pasatiempos tradicionales en experiencias digitales accesibles desde cualquier dispositivo.

### Lo que incluye

- **Sudokus**: Diferentes niveles de dificultad.

* **Crucigramas**: Temáticas variadas y actualizadas.
- **Sopas de letras**: Generación dinámica de tableros.

## Tecnologías

- **Lógica**: JavaScript ES6+
- **Diseño**: CSS Moderno / Flexbox
- **Generación**: Algoritmos de generación aleatoria de tableros.
