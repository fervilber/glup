# E-commerce Dashboard

Un panel de administración moderno para tiendas en línea construido con React y Tailwind CSS.

## Características

- **Gestión de Inventario**: Añadir, editar y eliminar productos.
- **Gráficos en Tiempo Real**: Visualización de ventas y usuarios activos.
- **Modo Oscuro**: Interfaz adaptable a preferencias del sistema.

## Tecnologías Utilizadas

- **Frontend**: React, Voucher, Tailwind CSS
- **Estado**: Redux Toolkit
- **Gráficos**: Chart.js
