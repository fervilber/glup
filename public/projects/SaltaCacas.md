# SaltaCacas

Nuestro primer juego digital.

## Resumen

Permite a los usuarios buscar el clima actual y el pronóstico de 5 días para cualquier ciudad del mundo.

> "La simplicidad es la máxima sofisticación."

Utiliza **Fetch API** para llamadas asíncronas y gestiona errores de red de forma elegante.

## Stack

- JavaScript Vanilla
- CSS Grid
- OpenWeatherMap API
