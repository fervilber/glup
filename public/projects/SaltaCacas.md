## Resumen

es el primer juego que hacemos y cuando mismo empiezas puedes elegir entre  3 personages distintos cada uno tiene un número, el objetivo es esquivar las cacas y coger los sucifientes puntos, para ganar puntos tienes que coger alimentos hay 2 fases apartir de un número de puntos pasas al 2 nivel.Tienes 3 vidas y cuando te chocas con las cacas se te quitan las vidas una por una así que cuando pierdes las vidas sale una pantalla que pone GAME OVER y se te guarda el proceso pones tu nombre y así se te guarda el record.

## Stack

Esta app tiene varias versiones, la primera la hicimos en python con pygame. En el 2025, queríamos hacer un juego online, todo en web, que no necesitara instalación de nada y lo conseguimos usando Phaser html y javascript.
